# Multifunctional-diagnostic-device-covid patients
Low-cost Arduino-based device for monitoring SpO2, heart rate, and temperature with Android app and voice alerts
## Overview
This project involves the design and development of a low-cost, multifunctional
diagnostic device to continuously monitor vital parameters such as heart rate,
blood oxygen saturation (SpO2), and body temperature.

## Hardware Components
- Arduino microcontroller
- MAX30100 pulse oximeter
- MLX90614 temperature sensor
- HC-05 Bluetooth module
- OLED display
- APR33A3 voice record module

## Features
- Real-time SpO2, heart rate, and temperature monitoring
- Android app connectivity via Bluetooth
- Voice alerts for abnormal temperature
- Low-cost and power-efficient design

## Applications
- COVID patient monitoring
- Home healthcare
- Wearable health monitoring
